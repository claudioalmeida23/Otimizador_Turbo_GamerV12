# 🚀 OTIMIZADOR TURBO GAMER v12

O **OTIMIZADOR TURBO GAMER v12** é uma ferramenta de automação em Batch desenvolvida para otimizar o Windows, focando na estabilidade do sistema e na redução de latência para jogos.

> **Desenvolvido por:** Claudio Almeida

---

## 🛑 AVISO IMPORTANTE (LEIA ANTES DE USAR)

**ESTE SCRIPT NÃO CRIA FPS.** Muitos usuários acreditam erroneamente que scripts podem aumentar o poder de hardware. Isso é um mito. 
* **O que o script faz:** Otimiza como o Windows gerencia os recursos que você já tem. Ele remove "lixo" do sistema, desativa rastreadores inúteis e prioriza o processamento para o jogo aberto.
* **O resultado:** Um sistema mais estável, com menos quedas bruscas de quadros (stuttering) e menor latência de rede.

---

## 🛠️ Principais Funções

### 1. Performance e CPU
* Ativa o plano de **Desempenho Máximo**.
* Ajusta o `Win32PrioritySeparation` para dar foco total ao jogo em primeiro plano.
* Ativa o **Game Mode** nativo do Windows.

### 2. Limpeza Profunda (Deep Clean)
* Limpa arquivos temporários e logs do sistema.
* Esvazia o **Cache de Shaders DirectX** (essencial para evitar travamentos).
* Esvazia a lixeira automaticamente.

### 3. Rede e Latência
* Otimiza parâmetros TCP para melhorar a estabilidade da conexão (Ping).
* Desativa a **Telemetria** (coleta de dados da Microsoft) para liberar memória e CPU.

### 4. Manutenção de Disco
* Limpa cache da **Windows Store** e arquivos antigos do **Windows Update** que podem ocupar vários GBs.

---

## 📋 Como Executar

1. Baixe o arquivo `TurboV11.bat`.
2. Clique com o botão direito e escolha **"Executar como Administrador"**.
   * *O acesso administrativo é necessário para alterar planos de energia e registros do sistema.*
3. Siga as instruções na tela e leia as descrições de cada função.

---

## 🛡️ Segurança
* **Ponto de Restauração:** O script oferece a opção de criar um ponto de restauração antes de aplicar mudanças.
* **Reversão:** Existe uma opção no menu para retornar o Windows ao modo "Equilibrado" original.
